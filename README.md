## Explanation for how to download
1. Open a folder i.e: ocarina lessons, then select a file i.e: ocarina lessons 1-4.7z
2. On the far right it should have a button: **Raw** should download

## Nuances:
If the file has a 001, 002 it means you must download both, then drag select both of them once they're on your pc and right click->extract
i.e: ocarina lessons 5-8.7z.001, ocarina lessons 5-8.7z.002
